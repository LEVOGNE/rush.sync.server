Quick Start

Create an index.html in www/{{SERVER_NAME}}-[{{PORT}}]/ to set your main page. It will be served at the root URL and file changes trigger Hot Reload automatically.

This dashboard is always available at /.rss/ — even when your index.html is active. You can also configure a custom 404 page and PIN protection in Settings.
